# WearWeb Backend

This is the backend API for the WearWeb project built using Node.js and Express.

## Features
- User Registration API
- Get All Users API
- MongoDB Database Connection

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

1. Clone the repository

git clone https://github.com/nensiKachhadiya26/wearweb_backend2026.git

2. Install dependencies

npm install

3. Run the server

npm start
